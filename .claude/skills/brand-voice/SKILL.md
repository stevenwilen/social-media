# Brand Voice — Veronica Wilen, Realtor

## Step 0 — Always load the profile first
Before writing anything, read **`brand/brand-profile.md`** (relative to the project root).
It holds Veronica's market, brokerage, license, audience, voice samples, hashtags, CTAs,
platform IDs, and compliance rules. Treat it as the source of truth. If a `[FILL IN]`
field you need is still blank, use the best-practice default AND note the gap to the user.

## What this skill guarantees
Any content you produce under this skill must be:
1. **On-voice** — warm, knowledgeable, approachable, confident, community-minded. Educates
   first, sells second. Never hype-y or salesy. (See profile §3; match the voice samples if present.)
2. **Value-forward** — lead with insight/story/local knowledge, not "call me to buy a house."
   Target ~70% value / 30% listings across a batch.
3. **Platform-shaped** — written to the right length/format for the target platform (profile §6).
4. **Compliant** — passes the Fair Housing + disclosure guardrails (profile §9). This is a hard gate.

## Writing procedure
1. **Confirm the target platform(s)** and the content pillar (profile §4). If unclear, ask or infer.
2. **Pick an angle** that fits the pillar and leads with value (a stat, a story, a local insight,
   a question) — not a listing dump.
3. **Draft to the platform's shape:**
   - **Facebook** — 3–6 short lines: hook → value/context → soft CTA. ≤3 hashtags. Emoji sparingly.
   - **Instagram** — line-1 hook (that's the feed preview) → short body → CTA → 8–15 targeted
     hashtags (end block or first comment). Needs a reel/image. More emoji OK. Reminder: IG on
     Blotato **requires media** — flag if none is provided.
   - **YouTube** — front-loaded title (≤~70 chars) + 2–3 sentence description + CTA + links.
4. **Rotate CTAs and hashtag sets** (profile §7–8) — never reuse the exact same set back-to-back.
5. **Run the compliance gate** (below). Rewrite anything that trips it.
6. **Present clearly** — group drafts by platform, number them, keep formatting clean so the user
   can quickly pick, edit, or approve for posting/scheduling.

## Compliance gate (check every draft before showing it)
Reject/rewrite if the copy:
- expresses a preference/limitation about people ("perfect for families," "safe area,"
  "great for retirees," religion/ethnicity references, etc.) — describe the **property**, not the buyer;
- guarantees price, timeline, or returns;
- shares listing data you're not authorized to post.
Add brokerage/license disclosure and "Equal Housing Opportunity" where the profile/state requires it.
